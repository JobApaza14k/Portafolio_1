Clase 1: Método de Newton-Raphson

Este programa calcula las raíces de una función mediante el método de Newton-Raphson.


def f(x):
    return x**3 - 4*x + 1

def f_derivada(x):
    return 3*x**2 - 4

x0 = 1
for i in range(10):
    x1 = x0 - f(x0)/f_derivada(x0)
    x0 = x1
print("Raíz aproximada:", round(x0, 4))

Clase 2: Método de Bisección

Ejemplo simple de cómo encontrar raíces mediante el método de bisección.


def f(x):
    return x**3 - 6*x + 2

a, b = 0, 2
for i in range(10):
    c = (a + b) / 2
    if f(a) * f(c) < 0:
        b = c
    else:
        a = c
print("Raíz aproximada:", round(c, 4))

Clase 3: Método de la Secante

Encuentra raíces sin necesidad de derivadas usando el método de la secante.


def f(x):
    return x**2 - 3

x0, x1 = 1, 2
for i in range(8):
    x2 = x1 - f(x1)*(x1 - x0)/(f(x1) - f(x0))
    x0, x1 = x1, x2
print("Raíz aproximada:", round(x2, 4))

Clase 4: Método de Punto Fijo

Reescribe la función f(x)=0 en forma x=g(x) para aplicar iteraciones sucesivas.


def g(x):
    return (x**3 + 1)/4

x0 = 1
for i in range(10):
    x1 = g(x0)
    x0 = x1
print("Raíz aproximada:", round(x0, 4))

Clase 5: Método de Gauss-Seidel

Resuelve sistemas de ecuaciones lineales por aproximaciones sucesivas.


x, y, z = 0, 0, 0
for i in range(10):
    x = (10 - y - z)/5
    y = (12 - 2*x - z)/4
    z = (15 - x - 2*y)/5
print("Solución aproximada:", round(x,2), round(y,2), round(z,2))
